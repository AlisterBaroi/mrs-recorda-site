## 快速幂

```cpp
ll Pow(ll x,ll y)
{
	x%=mod;
	ll res=1;
	while(y)
	{
		if(y&1)res=res*x%mod;
		x=x*x%mod;
		y>>=1;
	}
	return res;
}
```

## 快速乘

快速乘用于计算 $xy\bmod p$，可以防止直接计算 $xy$ 过大导致溢出。

```cpp
ll mul(ll x,ll y)
{
	x%=mod;
	ll res=0;
	while(y)
	{
		if(y&1)res=(res+x)%mod;
		x=(x+x)%mod;
		y>>=1;
	}
	return res;
}
```
