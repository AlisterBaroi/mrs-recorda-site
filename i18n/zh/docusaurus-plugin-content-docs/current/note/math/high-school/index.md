## 教材

人教版高中数学教材分为 A 版和 B 版。

- A 版分为 $2$ 册必修和 $3$ 册选修（选择性必修）。
- B 版分为 $4$ 册必修和 $3$ 册选修（选择性必修）。

GitHub 上有一个开源仓库 [ChinaTextbook](https://github.com/TapXWorld/ChinaTextbook/tree/master) 可以下载几乎所有小初高、大学的 PDF 教材。
